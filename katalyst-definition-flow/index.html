RoadItem State Machine
proposed → adr_validated → bdd_pending → bdd_complete ⬤ GO → implementing → nfr_validating → complete ✓
nfr_validating ↔ nfr_blocked (loop until resolved)  ·  Source: packages/foe-schemas/src/taxonomy/extensions/road-item.ts
Definition Gate: A Jira initiative is not ready to build until UserTypes, UserStories, Capabilities, and BDD Specs are status: approved and adrs.validated: true. The GO arrow fires when RoadItem.status = bdd_complete.
