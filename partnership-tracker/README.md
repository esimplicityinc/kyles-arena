# Partnership Tier Advancement Tracker

eSimplicity's 2026 Partnership Goals Dashboard for tracking advancement across AWS, Databricks, Salesforce, and Snowflake partner programs.

## Features
- Interactive progress tracking with persistent data
- Workback timeline through Dec 2026
- Risk log with severity levels
- Stakeholder engagement tracking
- Print/Export to PDF

## Deployment
This site is deployed via GitHub Pages.

## Usage
Open `index.html` in any browser, or visit the deployed URL.
